# 景区导览服务 AI 数字人

基于 AI 数字人技术的智能景区导览系统，为游客提供 7×24 小时在线的智能导游服务，同时为景区管理方提供数据看板。

## 项目概述

本项目针对景区传统导览服务的痛点（导游资源稀缺、无法互动、管理盲区），构建具备多模态交互能力的 AI 数字人导游系统。

### 核心能力

- **多模态交互**：支持语音输入和文本输入，数字人以语音、表情、口型同步的方式回答
- **智能问答**：基于 RAG 技术，准确回答景区历史、文化、景点特色等常见问题
- **个性化推荐**：根据游客兴趣推荐不同游览路线和讲解重点
- **数据分析**：为管理方提供游客反馈分析和服务质量报告

### 技术架构

```
前端 (Qt6 + QML)
    └── C++ 后端类：ApiService、LoginManager、ConversationManager 等

后端 (Python + FastAPI)
    └── RAG 服务（Chroma 向量存储 + bge-small-zh-v1.5 嵌入模型）
    └── LLM 服务（兼容 OpenAI 协议，配置为 DeepSeek V4 flash）

通信：HTTP + WebSocket
数据库：SQLite + Chroma
```

## 技术栈

| 层级       | 技术                                |
| ---------- | ----------------------------------- |
| 前端框架   | Qt 6.10+ / C++ / QML                |
| 后端框架   | Python / FastAPI (异步)             |
| 向量数据库 | Chroma                              |
| 嵌入模型   | BAAI/bge-small-zh-v1.5              |
| 大模型     | DeepSeek V4 flash (兼容 OpenAI API) |
